Học CSS Selector Nâng Cao

Tác giả: John Doe 15/01/2024 5 phút đọc

CSS Selector là công cụ mạnh mẽ giúp bạn chọn và tạo kiểu cho các phần tử HTML. Trong bài viết này, chúng ta sẽ khám phá các selector nâng cao.

Giới thiệu

Selector cơ bản bao gồm element, class và id selector.

Tuy nhiên, CSS còn nhiều selector phức tạp hơn.

Học đi đôi với hành, lý thuyết kết hợp thực tiễn.

Các loại Selector

Có nhiều loại selector khác nhau trong CSS.

Simple Selectors

  • Element selector
  • Class selector
  • ID selector
  • Universal selector

Combinators

  1. Descendant combinator (space)
  2. Child combinator (>)
  3. Adjacent sibling (+)
  4. General sibling (~)
.parent > .child { color: red; }

Bảng so sánh

Selector Ví dụ Độ ưu tiên
Element p 1
Class .text 10
ID #main 100
Attribute [type="text"] 10
